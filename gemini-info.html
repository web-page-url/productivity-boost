03

Core Conventions

Productivity Boost standardizes how AI capabilities are handled through a simple set of rules:

Universal Paths: Maps well-known disk locations for all major coding agents.

Metadata via SKILL.md: Uses standard Markdown and YAML frontmatter for tool definitions.

Atomicity: Toggling skills is done by physically moving folders to a hidden .disabled directory.
